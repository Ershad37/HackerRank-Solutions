# Description
This program calculates the sum of a very large array of integers.
It ensures that the sum can handle numbers larger than typical 32-bit integers.
# Problem Statement
Given an array ar of integers, compute the sum of all its elements and return the result as a long integer.
Example:
Input: [1000000001, 1000000002, 1000000003, 1000000004, 1000000005]
Output: 5000000015
