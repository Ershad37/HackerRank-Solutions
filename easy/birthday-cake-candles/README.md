# Description
This program determines how many candles on a birthday cake are the tallest.
It iterates through the list of candle heights to find the maximum and counts how many candles have that height.
# Problem Statement
You are given an array candles representing the height of each candle.
Return the number of candles that are tallest.
Example:
Input: [3, 2, 1, 3]
Output: 2
Explanation: The tallest candle has a height of 3, and there are 2 candles of that height.
