# Description
This program prints a right-aligned staircase of height n using the # symbol.
Each line contains increasing numbers of # characters, preceded by spaces to align the staircase to the right.
# Problem Statement
Given an integer n, print a staircase of height n composed of # symbols.
The staircase is right-aligned.
Each line contains spaces followed by # characters.
Example:
Input: n = 4
Output:
   #
  ##
 ###
####
