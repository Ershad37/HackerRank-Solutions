# Description
This program calculates the ratios of positive, negative, and zero elements in an integer array.
It prints each ratio with 6 decimal places of precision, as required in many competitive programming challenges.
# Problem Statement
Given an array of integers arr, calculate and print:
The fraction of elements that are positive
The fraction of elements that are negative
The fraction of elements that are zero
Each fraction should be printed on a new line with 6 decimal places.
Example:
Input: [1, 1, 0, -1, -1]
Output:
0.400000
0.400000
0.200000
Explanation:
Positive numbers: 2 / 5 = 0.4
Negative numbers: 2 / 5 = 0.4
Zeros: 1 / 5 = 0.2
